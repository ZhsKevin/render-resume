# 示例项目合集

这里收录了 render-resume 内置的虚构项目说明。它们用于展示如何把项目经历写得更像真实工程案例，但不包含任何真实个人、公司或业务信息。

## 项目列表

### [Orion AgentMesh｜企业级智能体编排平台](../../mdviewer.html?file=assets/markdown/agent-mesh.md)

- 面向企业流程自动化的多智能体协作平台，负责把复杂任务拆解为可执行、可追踪、可回放的步骤。
- 重点展示智能体编排、工具调用治理、权限隔离、审计回放和质量评测闭环。
- 适合体现 AI Native 应用从原型到平台化落地的完整工程思路。

### [Nebula RAG｜多模态知识检索与推理系统](../../mdviewer.html?file=assets/markdown/multimodal-rag.md)

- 面向企业知识管理的 RAG 系统，覆盖文本、表格、图片、网页和演示文稿等多类资料。
- 核心能力包括文档解析、混合检索、语义重排、引用溯源和答案质量评估。
- 适合展示如何降低大模型幻觉，并把知识问答做成可运营、可复核的产品能力。

### [FeatherServe｜边缘端大模型推理加速框架](../../mdviewer.html?file=assets/markdown/edge-inference.md)

- 面向边缘设备的轻量推理运行时，强调弱网、低延迟、低成本和离线可用。
- 支持模型量化、分层缓存、动态批处理、设备分组、灰度发布和离线降级策略。
- 适合展示在真实资源约束下部署 AI 能力时的系统设计与性能优化能力。

### [ApolloSim Lab｜自动驾驶场景仿真与评测平台](../../mdviewer.html?file=assets/markdown/autonomy-sim.md)

- 用于自动驾驶算法验证的场景仿真平台，面向长尾风险场景持续构造回归用例。
- 重点展示传感器回放、参数化场景生成、算法接入、指标评测和失败原因分析。
- 适合体现复杂算法系统中“可重复验证”和“可解释评测”的工程价值。

### [Helios Quant Lab｜实时风控与因子研究平台](../../mdviewer.html?file=assets/markdown/quant-lab.md)

- 面向量化研究的实时数据与风控平台，覆盖数据接入、因子计算、策略回测和告警解释。
- 通过批处理与流式计算结合，支持研究实验复现、风险监控和策略报告生成。
- 适合展示数据平台、实时计算、风控规则和业务分析之间的协同设计。

### [render-resume｜静态简历与作品集模板](../../mdviewer.html?file=assets/markdown/resume-pages.md)

- 本仓库本身的说明文档，介绍如何使用 Vite、JSON 和 Markdown 搭建静态简历页面。
- 模板内置虚构数据、项目详情页、响应式样式和 GitHub Pages 友好的构建流程。
- 适合快速发布公开个人主页，也适合作为前端静态页面练习项目继续扩展。

## 写作提示

好的项目介绍不必堆满术语。建议按“项目定位、核心能力、技术方案、示例成果、可展示亮点”来写，这样读者能快速理解你做了什么、为什么重要、难点在哪里。
