# Orion AgentMesh

> 企业级智能体编排平台，示例项目，全部信息均为虚构。

## 项目定位

Orion AgentMesh 是一套面向企业内部流程自动化的多智能体协作平台。它把大模型、工具调用、权限校验、流程引擎和审计系统组合在一起，让智能体不只是回答问题，而是可以在受控环境中完成任务拆解、资料检索、接口调用、审批流推进和结果回写。

平台适合用于研发助手、运营助手、数据分析助手、知识库问答、IT 服务台和销售线索整理等场景。

## 核心能力

- 多智能体编排：支持 Planner、Executor、Reviewer、Researcher 等角色协同完成复杂任务。
- 工具调用网关：统一管理内部 API、数据库查询、文档检索、工单系统和通知系统调用。
- 状态机运行时：把智能体执行过程拆成可观测、可重放、可中断的步骤，降低黑盒风险。
- 安全与审计：为每次工具调用记录操作者、输入、输出、权限上下文和模型版本。
- 评测闭环：支持离线任务集回放、人工标注、答案质量评分和线上灰度实验。

## 技术方案

平台采用“模型层 + 编排层 + 工具层 + 治理层”的架构。

模型层负责接入不同 LLM，并按任务成本、上下文长度和推理能力做动态路由。编排层用状态机描述任务执行过程，避免智能体无限循环或越权操作。工具层把企业内部系统封装成标准能力，治理层负责日志、追踪、权限、风控和评测。

## 示例成果

- 将人工整理周报、查询工单、生成需求初稿等流程收敛为可复用智能体任务。
- 通过离线回放发现高风险提示词和不稳定工具调用，提升上线前可控性。
- 为业务团队提供低代码配置界面，使非研发同学也能配置自己的部门助手。

## 可展示亮点

这个项目适合展示 AI Native 应用工程能力，重点不在“调用一个模型”，而在如何把模型能力工程化、平台化、可评估化。
