# Helios Quant Lab

> 实时风控与因子研究平台，示例项目，全部信息均为虚构。

## 项目定位

Helios Quant Lab 是一套面向量化研究和实时风控的工程平台。它把行情数据接入、因子计算、策略回测、实验追踪、风险预警和报告生成整合到统一工作流中，帮助研究团队减少重复开发，把更多精力放在策略假设和结果验证上。

## 核心能力

- 数据接入：支持多源行情、财务指标、新闻事件和另类数据统一落库。
- 因子计算：提供批处理与流式计算两种模式，支持因子版本管理。
- 策略回测：内置手续费、滑点、仓位约束、风险敞口和资金曲线分析。
- 实时风控：对异常波动、集中度、回撤和交易频率进行实时监控。
- 实验追踪：记录参数、数据区间、因子版本和回测结果，方便复现实验。

## 技术方案

平台采用数据湖、特征仓库和流式计算结合的设计。离线层负责历史数据清洗和回测，实时层负责行情消费、风险规则计算和告警推送。研究人员可以通过 Notebook 或 Web 控制台提交实验，也可以把稳定策略注册为可监控任务。

## 示例成果

- 将常用因子沉淀为可复用资产，避免每个研究员重复处理基础数据。
- 支持一键生成策略报告，包括收益、回撤、换手、暴露和稳定性分析。
- 在实时风控中加入异常解释，告警不只提示“出问题”，还给出可能原因。

## 可展示亮点

这个项目适合展示数据平台、实时计算和业务风控能力的结合，强调工程可靠性与研究效率的平衡。
